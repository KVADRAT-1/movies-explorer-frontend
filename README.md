# Проект «Movies explorer»

Сайт с авторизацией и регистрацией пользователей, возможностью изменять данные 
пользователя, поиска и добавления фильмов в избранное

## Технологии: 
* HTML 
* CSS 
* JS 
* React 
* Express.js 
* mongoDB
